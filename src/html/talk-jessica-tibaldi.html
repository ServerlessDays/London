Jessica Tibaldi

Jessica works in Microsoft as Technical Solutions Professional. Talk: Serverless for solving big issues... getting your Pizzas ASAP!

Wondering how to solve routing issues in a serverless way? This will be a practical demo on how to solve this problem with less infrastructure configuration as possible, using PaaS for messaging, maps and workflows capabilities provided by Azure Logic Apps and extended by Azure Functions.
